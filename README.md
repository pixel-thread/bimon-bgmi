## üèÜ Tournament Scoring System

Our platform implements the standard PUBG/BGMI tournament scoring:

- **Placement Points**: Based on final team ranking in each match
- **Elimination Points**: Points awarded for each player elimination (kill)
- **Total Score**: Combined placement and elimination points
- **K/D Calculations**: Automatic kill/death ratio tracking per PUBG standards
